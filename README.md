  <h1>Imagem Digital</h1>
    <h2>Definição</h2>
     <p>Imagem digital é a representação visual de dados que utiliza tecnologia para capturar, armazenar, processar, transmitir e exibir informações visuais. Ao contrario das imagens analógicas  que são registradas em meios físicos como por exemplo filmes fotograficos, . As imagens digitais são compostas por uma grade de elementos discretos chamados pixels, onde cada pixel contem um valor númerico que determina sua cor e intensidade. As imagens digitais são usadas em diversos contextos por exemplo fotografia digital, gráficos, mídia digital, medicina, ciência, entreterimento, etc. O fato das imagens serem digitais traz muitas vantagens permitindo seu armazenamento, transferência, impressão e reprodução. No geral há dois tipos de imagem digital, a imagem raster e a imagem vetorial</p>
